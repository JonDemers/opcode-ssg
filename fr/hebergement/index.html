Hébergement dans vos locaux

Si vous avez déjà toutes les infrastructures nécessaires pour héberger des applications, vous pouvez en tirer parti. Nous vous fournirons l'application que vous pourrez déployer dans votre environnement de production. Dans ce cas, vous serez responsable des tâches de maintenance comme la sauvegarde vos données (backup), les mises à jour systèmes, etc. En termes d'infrastructure, vous aurez besoin de:

  • Base de données (MySQL)
  • JVM (Oracle Java)
  • J2EE Web Container (Tomcat)
  • Nom de domaine, serveur DNS et certificat SSL
  • Nous recommendons aussi le serveur web Apache (comme proxy) et Ubuntu LTS
